# Chatbot Agent

## Setup

1. Install dependencies using [uv](https://github.com/astral-sh/uv):

    ```bash
    uv sync
    ```

2. Run the chatbot agent:

    ```bash
    python3 agent.py
    ```

## Requirements

- Python 3.x
- [uv](https://github.com/astral-sh/uv) installed

## Usage

After installing dependencies, start the chatbot with:

```bash
python3 agent.py
```
